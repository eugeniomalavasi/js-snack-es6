Creare un array di oggetti:
Ogni oggetto descriverà una bici da corsa con le seguenti proprietà: nome e peso.
Stampare a schermo la bici con peso minore utilizzando destructuring e template literal


prendiamo gli oggetti uno a uno, per ogni oggetto
    - se è il primo indice 
        -salviamo il  peso in una variabile
    - altrimenti se la variabile è minore del peso del successivo 
        - salva il peso 


    
quando abbiamo trovato il valore lo stampiamo in pagina
